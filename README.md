## Project Description

This is the official client repository, implementing the visual interface and user experience for the "Algo Que Pedir" app made for . It is a single-page application (SPA) built to communicate with a corresponding backend API.

## Technologies

* **Framework:** React
* **Language:** TypeScript
* **Build Tool:** Vite
* **Styling:** CSS
* **Linting/Formatting:** ESLint and Prettier
* **Testing:** Playwright (for End-to-End tests)

## Getting Started

### Prerequisites

Ensure you have Node.js (version specified in `.nvmrc` if present) and npm/yarn installed.

### Installation

1.  Clone the repository:
    ```bash
    git clone [https://github.com/MartinSchubert04/algoQuePedir-client.git](https://github.com/MartinSchubert04/algoQuePedir-client.git)
    cd algoQuePedir-client
    ```
2.  Install dependencies:
    ```bash
    npm install
    # or
    yarn install
    ```

### Development

To run the application

```bash
npm run dev
# or
yarn run dev
```
